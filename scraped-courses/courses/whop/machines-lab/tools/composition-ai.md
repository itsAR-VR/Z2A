# Composition AI

**Type:** Interactive AI Tool (Whisp Chatbot)
**URL:** https://whop.com/joined/machines-lab/composition-ai-HWpSeSHyGCg63n/app/

## Overview

Composition AI is an interactive chatbot interface designed to help users create perfect AI image prompts. It's built on Whop's Whisp platform.

## Features

- **AI Image Prompt Generation:** Helps users craft effective prompts for AI image generation tools
- **Nanobanana Integration:** Can generate images directly using Nanobanana
- **Chat History:** Saves conversation history for reference
- **Interactive Interface:** Real-time chat-based interaction

## How to Use

1. Navigate to the Composition AI page
2. Type your request in the text box (e.g., "What do you want to know?")
3. The AI will help you craft effective image prompts
4. Use the Nanobanana integration to generate images directly

## Interface Elements

- **Main Chat Area:** Central area for conversation with the AI
- **Text Input:** "What do you want to know?" prompt field
- **Send Button:** Submit your query
- **Nanobanana Button:** Generate images with integrated tool
- **Chat History Sidebar:** Access previous conversations

## Use Cases

- Crafting detailed prompts for Midjourney, DALL-E, Stable Diffusion
- Learning prompt engineering for image generation
- Getting feedback on existing prompts
- Exploring creative image concepts
