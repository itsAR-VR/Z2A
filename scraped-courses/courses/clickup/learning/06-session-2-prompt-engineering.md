# Session 2: Prompt Engineering

## 4 Types of Prompts:

1. **One-Shot Prompt**
2. **Chain of Thought**
3. **Decision Tree** [Jimmy's Second Favourite]
4. **Reverse Engineering** [Jimmy's Favourite]

## Prompt Components

1. **Identity** - Who the AI should be
2. **Task** - What the AI should do
3. **Context** - Background information
4. **Output** - Expected format/result
