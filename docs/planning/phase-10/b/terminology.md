## Status Terms (single source)
- **Canonical**: `Applications open now`
- Do not use: `Admissions open`, `Admissions live`

## Primary Action Terms
- **Primary CTA**: `Apply / Reserve Seat`
- **Mobile sticky CTA**: `Apply now`
- **Processing state**: `Redirecting...`

## Payment Mechanics Terms
- `Deposit` = payment made now (`$100`)
- `Remainder` = amount handled after check-in process
- `Checkout via Stripe` = secure payment flow
- `Full refund by end of Day 2` = canonical refund trust phrase

## Referral Language
- **Canonical label**: `Referral code (optional)`
- **Helper text**: `Referral codes help us track how you found us.`
- Do not imply guaranteed discount or payout if that is not current behavior.

## Audience Language Rules
- Prioritize plain professional language over technical jargon.
- Keep message ladder explicit:
  1) save time now,
  2) build practical skill,
  3) create upside with reclaimed time.

## Style Rules
- Use active voice.
- Keep each important statement one idea per sentence.
- Always pair policy statements with immediate user meaning.
