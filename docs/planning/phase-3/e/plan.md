# Phase 3e — Ops + GTM Assets (Agenda, Prework, Referrals, Outreach, Content Capture)

## Focus
Cover the operational deliverables required to run and scale the pilot, as specified in `Plans/plan.md`.

## Inputs
- `Plans/plan.md` (agenda/run-of-show, prework, referral program, outreach, content capture, post-event loop).

## Work
1. **Agenda / run‑of‑show**
   - 2‑day structure with teaching blocks → pod build time → demos.
2. **Prework checklist**
   - Environment setup, accounts, datasets, and readiness criteria.
3. **Pod workflow**
   - Pod worksheet template or task checklist.
4. **Referral program copy**
   - Referrer DM template + referred attendee message.
5. **Outreach sequences**
   - LinkedIn DM + email sequence (pilot).
6. **Content capture plan**
   - Shot list, testimonial prompts, event trailer outline.
7. **Post‑event loop**
   - Survey, testimonial harvesting, and LinkedIn post request.

## Output
- A minimal but complete ops asset pack ready to produce.

## Handoff
Pass to Phase 3f for QA/deploy alignment and ensure assets map back to site copy.
